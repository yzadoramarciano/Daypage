# Daypage

Aplicação web de diário digital desenvolvida com HTML, CSS e JavaScript.

Funcionalidades:
- Cadastro de usuários
- Login
- Armazenamento de notas por usuário
- Criação de novas páginas
- Exibição das notas em cards
- Pop-ups personalizados
- Interface responsiva

Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript
- LocalStorage

Objetivo:
Projeto desenvolvido para praticar desenvolvimento front-end, manipulação do DOM e armazenamento de dados no navegador.
